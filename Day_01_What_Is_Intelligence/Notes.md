# Day 1: What is Intelligence (Human vs Artificial)

## Core Idea

Intelligence is not just "thinking"; it is a continuous loop of:

1. **Perception**: Observing the environment.
2. **Reasoning**: Processing information based on rules or patterns.
3. **Action**: Implementing a change in the environment.
4. **Learning**: Updating behavior based on the outcome (feedback).

## The Intelligence Loop

- **Perception**: Sensors, data input, visuals.
- **Reasoning**: Logical IF-THEN rules, mathematical models, neural processing.
- **Action**: Actuators, API calls, text generation.
- **Learning**: Backpropagation, reinforcement, manual threshold adjustment.
