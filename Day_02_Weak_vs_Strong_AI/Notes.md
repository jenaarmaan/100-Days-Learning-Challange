# Day 2: Weak AI vs Strong AI (Narrow vs General)

## Core Idea

- **Weak/Narrow AI**: Performs extremely well on a specific task (e.g., chess, image recognition) but fails completely if the input shifts even slightly outside its training scope. Most modern "AI" falls into this category.
- **Strong/General AI (AGI)**: Possesses the ability to understand, learn, and apply knowledge across multiple domains, similar to human intelligence.

## Intuition

A Narrow AI is like a expert calculator for a specific formula. If you change the variables or the context, it provides an incorrect answer because it "knows" only the pattern it was shown.
